动手实操 - 在本地通过ComfyUI生成图片

十二亚晖 2025-10-05 09:00:00
Categories： AI Tags： AI 技术 实操

这份实操文档专为你的 T1000 (1GB 显存) 硬件环境优化，旨在帮助你从零构建一个性能稳定、具备电影感画质的 AI 创作站。

第一阶段：环境搭建（Stability Matrix 与 ComfyUI）

1. 安装管理大本营：Stability Matrix

  • 下载与放置：将 StabilityMatrix-win-x64.zip 解压到一个空间充足的非系统盘（如 D 盘）。
  • 启动：运行 StabilityMatrix.exe，它会自动检测环境（Git、Python 等）并帮你安装好基础组件。

2. 安装 ComfyUI 软件包

  • 在 Stability Matrix 左侧点击 Packages（软件包）。
  • 点击 Add Package，选择 ComfyUI 进行安装。

    提示：安装完成后，Stability Matrix 会帮你管理所有复杂的路径，你只需要点击“运行”即可。

第二阶段：模型下载与归类

为了实现写实风格，你需要下载以下三类文件。请在 Model Browser 中勾选 SD 1.5 标签后搜索：

  1. 大模型 (Checkpoint)：搜索 Realistic Vision 或 EpicRealism。
  • 存放路径：Data\Models\Checkpoints
  1. 色彩修复 (VAE)：搜索 vae-ft-mse-840000-ema-pruned。
  • 存放路径：Data\Models\VAE
  1. 细节插件 (LoRA)：搜索 More Details 或 Detail Tweaker。
  • 存放路径：Data\Models\Loras

第三阶段：构建“电影感”进阶工作流

请按照以下顺序在 ComfyUI 画布上连接节点，这是针对 1GB 显存最稳妥的方案：

1. 核心链路 (Checkpoint -> LoRA -> Sampler)

  • Load Checkpoint：选择你的写实大模型。

  • Load LoRA：

  • MODEL 接 Checkpoint 的 MODEL；CLIP 接 Checkpoint 的 CLIP。

  • 强度（strength）建议设为 0.6。

  • KSampler (关键参数)：

  • model 接 LoRA 的 MODEL。

  • cfg: 7.0（切记不要设太高或太低）。

  • sampler: dpmpp_2m / scheduler: karras。

  • steps: 20-25 步。

2. 提示词链路 (CLIP -> Text Encode)

  • 正向提示词：从 LoRA 的 CLIP 出口连接。输入：portrait of a woman, cinematic lighting, 8k, highly detailed skin。
  • 负向提示词：同样从 LoRA 的 CLIP 出口连接。输入：bad anatomy, blurry, cartoon, lowres。

3. 色彩解码链路 (VAE -> Image)

  • Load VAE：选择下载的 840000 修复模型。

  • VAE Decode：

  • samples 接采样器的 LATENT。

  • vae 必须接 Load VAE 节点的输出（不要接大模型自带的 VAE）。

  • Save Image：查看最终成品。

第四阶段：1GB 显存避坑守则

  • 尺寸控制：始终保持在 512x512。
  • 显存优化：在 Stability Matrix 的 ComfyUI 设置中，勾选 --lowvram 启动参数。
  • 模型选择：严禁加载 SDXL 或 Flux 模型，否则会瞬间崩溃。

第五阶段：二次启动操作手册（电脑关机后如何重开）

当你下次打开电脑，想要继续创作时，请严格执行以下步骤：

  1. 第一步：启动管理器
  • 双击运行 StabilityMatrix.exe。
  1. 第二步：进入软件包管理
  • 点击左侧菜单栏的 Packages 图标。
  1. 第三步：启动 ComfyUI
  • 找到 ComfyUI 卡片，点击右下角的 Launch（启动）按钮。
  • 此时会弹出一个黑色代码窗口，请不要关闭它，这是 AI 的大脑。
  1. 第四步：打开操作界面
  • 黑色窗口运行片刻后，Stability Matrix 通常会自动弹出浏览器窗口。
  • 如果没有自动弹出，请手动在浏览器输入：http://127.0.0.1:8188。
  1. 第五步：恢复工作流
  • 如果你上次关机前保存了工作流，它会自动加载。
  • 如果没有，你可以直接将上次生成的图片文件拖入浏览器窗口，ComfyUI 会自动还原当时所有的节点和参数。
