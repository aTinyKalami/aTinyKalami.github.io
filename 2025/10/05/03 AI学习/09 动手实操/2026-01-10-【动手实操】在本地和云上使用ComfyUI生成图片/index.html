动手实操 - 在本地和云端通过ComfyUI生成图片和视频

十二亚晖 2025-10-05 09:00:00
Categories： AI Tags： AI 技术 实操

一、在本地环境安装CompyUI，学习SD模型生成图片

本地机器显存有限（1GB），仅能用来学习基础的软件操作、工作流编辑，以及生成512X512大小以下的图片。

第一阶段：环境搭建（Stability Matrix 与 ComfyUI）

1. 安装管理大本营：Stability Matrix

下载 StabilityMatrix-win-x64.zip，解压并运行 StabilityMatrix.exe，它会自动检测环境（Git、Python 等）并帮你安装好基础组件。

2. 安装 ComfyUI 软件包

在 Stability Matrix 左侧点击 Packages（软件包）。
点击 Add Package，选择 ComfyUI 进行安装。
安装完成后点击Launch。

第二阶段：模型下载与归类

为了实现写实风格，需要下载以下三类文件。在 Model Browser 中勾选 SD 1.5 标签后搜索：

  1. 大模型 (Checkpoint)：搜索 Realistic Vision 或 EpicRealism。
    存放路径：Data\Models\Checkpoints

  2. 色彩修复 (VAE)：搜索 vae-ft-mse-840000-ema-pruned。
    存放路径：Data\Models\VAE

  3. 细节插件 (LoRA)：搜索 More Details 或 Detail Tweaker。
    存放路径：Data\Models\Loras

第三阶段：构建“电影感”进阶工作流

按照以下顺序在 ComfyUI 画布上连接节点：

1. 核心链路 (Checkpoint -> LoRA -> Sampler)

Load Checkpoint：选择你的写实大模型。
Load LoRA：根据不同目标，加载不同的LoRA效果模型。
MODEL 接 Checkpoint 的 MODEL；CLIP接 Checkpoint 的 CLIP。
强度（strength）建议设为 0.6。

KSampler (关键参数)：
model 接 LoRA 的 MODEL。
cfg: 7.0（切记不要设太高或太低）。
sampler: dpmpp_2m / scheduler: karras。
steps: 20-25 步。

2. 提示词链路 (CLIP -> Text Encode)

正向提示词：从 LoRA 的 CLIP 出口连接。输入：portrait of a woman, cinematic lighting, 8k, highly detailed skin。
负向提示词：同样从 LoRA 的 CLIP 出口连接。输入：bad anatomy, blurry, cartoon, lowres。

3. 色彩解码链路 (VAE -> Image)

Load VAE：选择下载的 840000 修复模型。
VAE Decode：
samples 接采样器的 LATENT。
vae 必须接 Load VAE 节点的输出（不要接大模型自带的 VAE）。

Save Image：查看最终成品。

第四阶段：1GB 显存避坑守则

  • 尺寸控制：始终保持在 512x512。
  • 显存优化：在 Stability Matrix 的 ComfyUI 设置中，勾选 --lowvram 启动参数。
  • 模型选择：严禁加载 SDXL 或 Flux 模型，否则会瞬间崩溃。

第五阶段：二次启动操作手册（电脑关机后如何重开）

当你下次打开电脑，想要继续创作时，请严格执行以下步骤：

  1. 第一步：启动管理器
  • 双击运行 StabilityMatrix.exe。
  1. 第二步：进入软件包管理
  • 点击左侧菜单栏的 Packages 图标。
  1. 第三步：启动 ComfyUI
  • 找到 ComfyUI 卡片，点击右下角的 Launch（启动）按钮。
  • 此时会弹出一个黑色代码窗口，请不要关闭它，这是 AI 的大脑。
  1. 第四步：打开操作界面
  • 黑色窗口运行片刻后，Stability Matrix 通常会自动弹出浏览器窗口。
  • 如果没有自动弹出，请手动在浏览器输入：http://127.0.0.1:8188。
  1. 第五步：恢复工作流
  • 如果你上次关机前保存了工作流，它会自动加载。
  • 如果没有，你可以直接将上次生成的图片文件拖入浏览器窗口，ComfyUI 会自动还原当时所有的节点和参数。

二、利用云侧算力平台，进行图像和视频生成

既然你已经有了一台 1GB 显存的本地电脑，最聪明的办法是**“本地设计，云端渲染”**：

本地构思：在本地打开 ComfyUI，虽然不出图，但可以把所有的节点（Load Checkpoint, KSampler, AnimateDiff 等）连好。

导出 JSON：点击右侧菜单的 「Save」，导出一个小小的 .json 工作流文件。

上云开机：在 AutoDL 租一台 RTX 3090（大约一两块钱一小时）。

上传并运行：通过浏览器把你的 JSON 文件拖进云端的 ComfyUI。

关机释放：视频生成完后，把结果下载回本地，立刻关机（释放实例），这样你可能只花了 5 毛钱就完成了本地电脑一辈子也跑不出来的视频。

在 AutoDL 中，你创建的每一个“实例”（也就是那台云电脑）都有两种状态：
状态
显卡 (GPU)计费适用场景正常开机处于激活状态全额扣费 (如 ￥1.5/时)跑图、生成视频、高负载测试。
无卡模式开机暂时移除 GPU极低扣费 (约 ￥0.1/时)开发工作流、整理模型文件、写代码。

第一阶段：本地预研 (￥0)
在你自己的 T1000 本地电脑上打开 ComfyUI。

尽管它跑不动，但你可以把节点先摆好、线连好。

点击 Save 导出工作流 JSON 文件。

第二阶段：云端无卡部署 (约 ￥0.1/时)
在 AutoDL 选择 「无卡模式」 启动你的实例。

打开云端的 ComfyUI 界面。

主要任务：上传你的 JSON 文件，并利用云端的高速带宽把工作流需要的 Checkpoint、LoRA、ControlNet 模型全部下载好。

这个阶段你不需要显卡，每小时只要一毛钱。

第三阶段：关机切换，瞬间超频 (开始扣全额)
在 AutoDL 控制台先 「关机」。

选择 「切换镜像/硬件」，把模式换成 「正常开机」（这时会带上 3090/4090 显卡）。

点击 Queue Prompt：让显卡全力输出，生成视频。

这个阶段按全额计费，但因为你已经提前准备好了，显卡一上来就干活，不浪费一秒钟。

第四阶段：成果回收
视频生成完后，通过文件管理器把视频下载到本地。

立刻点击「关机并释放」（如果不释放，硬盘存储可能还会产生微量费用）。

  1. 特别提醒：别忘了「释放」
    很多新手最容易踩的坑是：关机不等于停止扣费。

关机 (Shutdown)：停止计算计费，但会保留你的硬盘数据，系统会收取少量的硬盘存储费。

释放 (Terminate/Release)：彻底删除这台机器，停止所有扣费（但你的数据也没了，下次用要重装）。

三、ComfyUI的使用学习

可以在ComfyUI Wiki网站学习相关节点的功能。

ComfyUI 核心节点速查手册 (初学者必藏)

1. 加载类 (Loaders)

节点名称 功能说明 核心参数/注意事项
Load Checkpoint 加载大模型 (Base Model) 所有工作流起点，输出 MODEL, CLIP, VAE。
Load VAE 加载色彩修复模型 解决图片发灰问题（如 840000 模型）。
Load LoRA 加载风格/人物模型 用于固定特定人物或画风。

2. 编码类 (Encoding)

节点名称 功能说明 核心参数/注意事项
CLIP Text Encode 提示词输入框 分为 Positive (正向) 和 Negative (负向)。
VAE Encode 像素转潜空间 (图生图) 将已有图片转为 AI 可处理的 Latent 数据。

3. 采样与解码 (Sampling & Decoding)

节点名称 功能说明 核心参数/注意事项
KSampler 核心大脑 (采样器) 控制步数 (steps)、引导系数 (cfg) 和算法。
VAE Decode 潜空间转像素 将 AI 的计算结果还原成肉眼可见的图像。
VAE Decode (Tiled) 分块解码 (低显存必选) (1GB显存推荐) 防止大图生成时显存溢出。

4. 图像处理 (Image)

节点名称 功能说明 核心参数/注意事项
Save Image 最终导出 保存图片至 output 文件夹。
Upscale Image 图像放大 提升分辨率，通常接在 VAE 解码之后。

💡 颜色连线规则

  • 紫色 (MODEL): 传导模型权重
  • 黄色 (CONDITIONING): 传导提示词/控制信息
  • 粉色 (LATENT): 传导 AI 潜空间数据（看不见的图）
  • 青色 (IMAGE): 传导像素级图片（看得见的图）
  • 红色 (VAE): 传导编解码器信息

连线逻辑小贴士 (Tips)
颜色对应：ComfyUI 遵循“颜色匹配”原则。紫色接紫色 (MODEL)，黄色接黄色 (CONDITIONING)，青色接青色 (IMAGE)，红色接红色 (VAE)。

潜空间 (Latent)：看到粉色的连线 (LATENT) 代表数据还在 AI 的“脑子里”，必须经过 VAE Decode 转换成青色连线 (IMAGE) 你才能看到图。

右键万能：如果你找不到某个节点，在画布空白处双击左键，直接输入中文或英文关键词搜索即可。
