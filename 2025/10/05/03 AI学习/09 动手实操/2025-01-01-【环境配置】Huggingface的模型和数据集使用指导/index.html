Huggingface平台的模型和数据集使用

十二亚晖 2025-10-05 09:00:00
Categories： AI Tags： AI 实操 配置

Hugging Face 已成为 AI 开发者的必备工具，它提供了丰富的预训练模型、数据集等，帮助我们快速进行模型训练与推理。

将HuggingFace的模型和数据集下载到本地

经常会遇到需要将Huggingface上的模型或数据集下载到本地进行处理，或者再上传到其他平台的操作。

将模型和数据集下载到本地有多种方式：

1.Hugging Face CLI 下载：适合直接下载模型文件，无需写代码。
2.Python 代码下载：将CLI命令写入一个Python的.py文件，可供执行时调用，适合在项目中动态加载和缓存模型。
3.不通过Python和Pip，直接通过Git的命令行进行下载。

上面第1种和第2种方式其实区别不大，只不过第2种将CLI命令写进了一个Python的文件，然后执行文件得到结果，类似于DOS的批处理。好处是可以在全码开发时由程序调用Python文件来触发动态加载。

方法一：使用 Hugging Face CLI 下载模型

第 1 步：安装Python

访问 https://www.python.org/downloads/，下载 Python 最新版本。
安装时注意如下的重要设置：
    ✅ 勾选 "Add Python to PATH"
    ✅ 选择 "Customize installation"
    ✅ 确保 "pip" 被选中安装
    ✅ 建议勾选 "Install for all users"

第 2 步：安装 Hugging Face CLI 工具

本机打开 命令行（cmd 或 PowerShell），输入以下命令进行安装：

pip install huggingface_hub

说明：

  1. 如果运行pip，系统提示’pip’ 不是内部或外部命令，也不是可运行的程序或批处理文件。 那需要检查上一步安装的时候是否勾选pip。同时安装时如提前打开了cmd，安装完后需要重新打开cmd以便安装Python的系统环境变量顺利加载。
  2. huggingface_hub 模块包含 huggingface-cli，用于从 Hugging Face 上通过命令行来下载文件。

第 3 步：（可选步骤）登录 Hugging Face 账户（仅限下载私有模型和数据集）

如果你要下载 私有模型和数据集 或使用高权限 API 访问，需要登录 Hugging Face 账户。在命令行输入命令：

huggingface-cli login

获取 Access Token：

  1. 登录 Hugging Face 账户页面。
  2. 点击用户头像进入用户中心
  3. 点击Settings
  4. 点击AccessTokens
  5. 点击Create New Token，生成一个具有 Read 权限的 Token。

然后在 CLI 提示中粘贴 Token，然后执行。

HuggingFace模型使用各步骤说明

1. 安装与环境配置

步骤 说明
安装Python - 根据你的操作系统，安装最新版本的Python（https://www.python.org/downloads/）
- 建议使用Anaconda或Miniconda进行管理，（开发者需要在不同的版本/环境或支持库间切换。）
安装Hugging Face库 在终端cmd命令提示符中运行以下命令：pip install transformers
安装Hugging Face Hub CLI 运行以下命令：pip install huggingface-hub。这将安装Hugging Face Hub的命令行工具，方便您管理和分享您的模型和数据集。

2. 模型选择与加载

步骤 说明
选择模型库 根据您的任务需求，选择合适的模型库。Hugging Face提供了各种预训练模型，如BERT、GPT系列、T5等。
加载模型 使用transformers库中的AutoModel类，可以自动加载相应的预训练模型。例如：from transformers import AutoModelForSequenceClassification。
下载模型权重 如果您需要特定任务的预训练权重，可以使用transformers库中的download_pretrained函数进行下载。例如：transformers.download_pretrained(‘bert-base-uncased’)。

3. 数据预处理

步骤 说明
导入数据集 根据您的任务需求，导入相应的数据集。可以使用CSV、JSON等格式，也可以从Web上下载数据集。
数据清洗与处理 对数据进行清洗和处理，包括去除无关信息、处理缺失值、进行文本编码等。可以使用Pandas等数据处理库进行操作。
分割数据集 将数据集分为训练集、验证集和测试集，以便进行模型训练和评估。可以使用Scikit-learn等库进行操作。

4. 模型训练与评估

步骤 说明
准备数据输入 使用transformers库中的InputExample类准备训练和验证的数据输入。例如：from transformers import InputExample。
定义训练参数 根据您的任务需求，定义训练参数，如学习率、批大小等。可以使用transformers库中的TrainingArguments类进行设置。例如：from transformers import TrainingArguments。
训练模型 使用transformers库中的Trainer类进行模型训练。例如：from transformers import Trainer, AutoModelForSequenceClassification。在训练过程中，您可以选择使用GPU或TPU加速计算。
评估模型性能 使用测试集对训练好的模型进行评估，计算准确率、损失等指标，以便了解模型的性能表现。可以使用transformers库中的相关函数进行评估。例如：from transformers import evaluate, AutoTokenizer, AutoModelForSequenceClassification。在评估过程中，您可以选择使用GPU或TPU加速计算。
保存与分享模型 如果您希望保存或分享您的模型，可以使用Hugging Face Hub进行管理。将模型权重保存到Hub上，并生成相应的分享链接或二维码，以便其他人可以轻松地使用和评估您的模型。
